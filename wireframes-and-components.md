## Responsive Design Considerations

### Mobile-First Approach
- **Minimum touch target**: 44px (iOS) / 48dp (Android)
- **Font sizes**: Minimum 16px for body text
- **Contrast ratio**: Minimum 4.5:1 for normal text
- **Voice button**: Extra large (80px+) for easy access

### Tablet Adaptations
- **Two-column layout** for dashboard
- **Side-by-side** negotiation and coaching panels
- **Larger price visualization** components

### Accessibility Features
- **Screen reader support** for all interactive elements
- **High contrast mode** for bright outdoor conditions
- **Voice-only navigation** for hands-free operation
- **Haptic feedback** for button presses and confirmations
