A voice-first AI-powered web platform designed to empower local vendors in Indian mandis by removing language barriers, enabling fair price discovery, and supporting ethical negotiation.

## Problem
Local vendors often suffer losses due to:
- Language barriers
- Lack of real-time price awareness
- Weak negotiation power
- Dependence on middlemen

## Solution
Multilingual Mandi uses Generative AI to provide:
- Real-time multilingual voice translation
- AI-driven price discovery
- AI-assisted negotiation support
- Transparent price justification for buyers

- Generative AI for translation, negotiation, and price intelligence
